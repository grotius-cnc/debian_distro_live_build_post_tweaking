
# To show drives :

	$ lsblk

# The sdb is the usb.
# Copy iso file to usb :

	$ sudo dd bs=4M if=/yourfilepath/live-image-amd64.hybrid.iso of=/dev/sdb conv=fdatasync status=progress


	# You can check the bootable USB drive works by rebooting your computer and booting from the USB drive, 
	# or you can try booting from it in another computer.

	# You now have a portable working copy of Ubuntu or another Linux distribution of your choice. 
	# It will be pristine every time you boot it, and you can boot it on practically any PC you like.

	# Reference : https://www.howtogeek.com/414574/how-to-burn-an-iso-file-to-a-usb-drive-in-linux/
